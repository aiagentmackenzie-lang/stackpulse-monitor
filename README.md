# StackPulse Monitor

A native health-monitoring dashboard that integrates OSV database, endoflife.date API, and GPT-4o analysis to provide developers with a real-time Stack Health Score.

## Features

- **Vulnerability Tracking**: Monitor package vulnerabilities via OSV database
- **End-of-Life Alerts**: Track dependency lifecycle status
- **AI-Powered Analysis**: GPT-4o provides actionable insights
- **Stack Health Score**: Real-time metric for your tech stack health
- **GitHub Integration**: Connect repos for automatic monitoring

## Tech Stack

- Swift
- SwiftUI
- OSV API
- endoflife.date API
- OpenAI GPT-4o
